# PDF to Presentation Converter - User Guide

## Quick Start Guide

### For Non-Technical Users

1. **Double-click** the `PDF_to_Presentation_Converter.exe` file
2. Wait for the application to start (you'll see a command window)
3. Your web browser will automatically open to the application
4. If the browser doesn't open automatically, go to: `http://localhost:8501`

### How to Use the Application

1. **Upload a PDF**: Click "Browse files" and select your PDF document
2. **Choose Language** (Optional): Select "Turkish" if you want the presentation translated
3. **Convert**: Click "Convert PDF to Presentation"
4. **Wait**: The process may take a few minutes depending on the PDF size
5. **Download**: When complete, click "Download Presentation" to save the .pptx file

### What the Application Does

- **Extracts** text and images from your PDF
- **Creates** professional presentation slides using AI
- **Translates** content to Turkish (if selected)
- **Generates** a PowerPoint (.pptx) file ready for use

### Troubleshooting

**If the application doesn't start:**
- Make sure the `.env` file is in the same folder as the .exe file
- Check that your internet connection is working
- Try running the .exe file as administrator

**If the browser doesn't open:**
- Manually open your web browser
- Go to: `http://localhost:8501`

**If the conversion fails:**
- Check that your PDF file is not corrupted
- Make sure the PDF is not password-protected
- Try with a smaller PDF file first

### System Requirements

- Windows 10 or later
- Internet connection (for AI processing)
- At least 4GB RAM
- 500MB free disk space

### Files You Need

1. `PDF_to_Presentation_Converter.exe` - The main application
2. `.env` - Configuration file (contains your Azure credentials)

Both files must be in the same folder!
