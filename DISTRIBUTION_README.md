# PDF to Presentation Converter - Distribution Package

## For Your Father (Non-Technical User)

### What's Included:
1. `PDF_to_Presentation_Converter.exe` - The main application (107MB)
2. `.env` - Configuration file (contains your Azure credentials)
3. `USER_GUIDE.md` - Simple user guide

### How to Use:

1. **Create a new folder** on your father's computer (e.g., "PDF Converter")
2. **Copy these files** into that folder:
   - `PDF_to_Presentation_Converter.exe`
   - `.env` (make sure this contains your Azure credentials)
   - `USER_GUIDE.md` (optional, for reference)

3. **Double-click** `PDF_to_Presentation_Converter.exe` to run

4. **The application will:**
   - Open a command window
   - Start the web interface
   - Automatically open your browser to the application

### Important Notes:

- **Both files must be in the same folder** - the .exe and .env file
- **Internet connection required** - for AI processing
- **Windows 10 or later** - required
- **No installation needed** - just double-click and run

### Troubleshooting:

If the application doesn't start:
- Make sure both files are in the same folder
- Check that the .env file contains your Azure credentials
- Try running as administrator
- Check your internet connection

### File Sizes:
- Executable: ~107MB (includes all dependencies)
- .env file: ~1KB (just configuration)
- Total package: ~107MB

### Security Note:
The .env file contains your Azure API credentials. Keep it secure and don't share it publicly.
